What is Tech Consulting?

The Class Of Technology Consulting Program

The Class Of Technology Consulting program helps recent college graduates quickly design, develop, and deliver Oracle technology solutions in Oracle Cloud. Attendees also ensure that these solutions are delivered using Oracle's Total Cloud methodology, and adhere to best practices that have been defined from real-world consulting engagements. The program includes a mix of elearning, self-study, instructor-led workshops, and shadowing of active consulting engagements.

The Tech Consultant Role

As a member of a project team, you will follow standard practices and procedures to analyze requirements and provide quality work products to deliver functional and technical solutions for applications and technology implementations to meet customer specifications. You are expected to work directly with customers and members of the consulting delivery and sales teams on key activities such as:
  • Performing basic database administration and migration tasks
  • Building and executing on repeatable cloud solutions
  • Performing basic system administration and cloud infrastructure tasks
  • Assisting with creating new industry-focused solution innovations
  • Building cloud native applications using modern development technologies, including Microservices, Containers, and DevOps
  • Automating infrastructure as code
  • Work with a consulting team during requirements gathering, solution design, implementation, testing and production go live
  • Be actively involved in helping clients leverage Oracle Cloud solutions for application integration, process automation, and application development
  • Assisting with implementing Oracle cloud security solutions for identity and access management (IAM), threat detection, predictive analytics, security configuration management, and automated incident response (CASB), monitoring and analytics (SMA), database security and configuration and compliance (OCC) cloud service
  • Designing and building a modern user experience

Tech Consultant Qualifications

Preferred Qualifications
  • Bachelor's degree in Computer Science, Computer Engineering, Information Technology, Management Information Systems or equivalent experience
  • 0-2 years of overall experience in relevant functional or technical area
  • Demonstrates competence in a technology area
  • Ability to communicate effectively and build rapport with team members and clients.
Exposure to, or working knowledge in, any of the following areas:
  • Oracle database, SQL, PL/SQL, basic database administration tasks, migrating databases to cloud environments
  • Agile, SCRUM, SDLC, DevOps, rapid and iterative technical design/implementation based projects
  • Programming and scripting languages including: Javascript, Java, SQL, Node.js, Python, CSS, HTML5, Jenkins, Chef, JSON, Ansible, etc.
  • Cloud computing platforms, Saas Applications, development and/or integration, networking and firewalls
  • Hands on cloud experience with Oracle Cloud Infrastructure, AWS, Azure or Google
  • Software defined networking, network services such as DNS, NTP, NFS, LDAP, and/or Cisco, Juniper, Palo Alto, or F5 network devices
  • Linux or Windows System Administration
  • Experience with Git/GitHub
  • Application or Web security, OS security, Network Security, Infrastructure security
  • Experience with AI/ML, Analytics, or ETL Tools
