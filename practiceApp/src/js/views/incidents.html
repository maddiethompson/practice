What is tech consulting?

As a member of a project team, you will follow standard practices and procedures to analyze requirements and provide quality work products to deliver functional and technical solutions for applications and technology implementations to meet customer specifications. You are expected to work directly with customers and members of the consulting delivery and sales teams on key activities such as:

Performing basic database administration and migration tasks
Building and executing on repeatable cloud solutions
Performing basic system administration and cloud infrastructure tasks
Assisting with creating new industry-focused solution innovations
Building cloud native applications using modern development technologies, including Microservices, Containers, and DevOps
Automating infrastructure as code
Work with a consulting team during requirements gathering, solution design, implementation, testing and production go live
Be actively involved in helping clients leverage Oracle Cloud solutions for application integration, process automation, and application development
Assisting with implementing Oracle cloud security solutions for identity and access management (IAM), threat detection, predictive analytics, security configuration management, and automated incident response (CASB), monitoring and analytics (SMA), database security and configuration and compliance (OCC) cloud service
Designing and building a modern user experience
